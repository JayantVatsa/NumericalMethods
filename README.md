# NumericalMethods
This repository contains implementations of fundamental numerical methods commonly used in physics that I have learnt in my time as undergrad.

Methods implemented include:

- Curve Fitting (Least Square Regression)
- Interpolation (Lagrange)
- Root finding (Bisection, Newton-Raphson, Secant, False Position)
- Numerical integration (Trapezoidal, Simpson’s rule, Gauss Quadrature)
- Numerical solutions of differential equations (Euler, Modified-Euler, Runge-Kutta methods)
- Boundary Value Problems (Finite Difference Method (FDM), Shooting Method, Crank Nicolson Method)

All implementations are written in Python using basic scientific libraries.
